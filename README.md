Repo del sitio web de la 'Iglesia Bautista Fe' de Fort Worth, Texas, USA.

## Project Setup

```sh
yarn
```

### Compile and Hot-Reload for Development

```sh
yarn dev
```

### Compile and Minify for Production

```sh
yarn build
```
